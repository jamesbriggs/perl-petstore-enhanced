Petstore Enhanced Sample Client Programs

After the server is listening, you can send it requests using the included client programs.

Steps:

1. source ../set.sh
2. ./curl_client_sample.sh
3. ./perl_client_sample.pl
4. ./python_client_sample.py
5. ./ruby_client_sample.py
6. ./wget_client_sample.sh 

Take a look at the source code to see how to write your own programs.

These are non-trivial programs with features such as HTTP basic authentication and timeout.
